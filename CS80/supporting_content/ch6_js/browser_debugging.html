Browser Support for Debugging

Whenever the JavaScript interpreter in the browser encounters a syntax or runtime error while executing your script, it will notify you by producing an error message. It is extremely important and useful to study these error messages when debugging your script. You may need to adjust your browser's settings to see these messages. How you do this depends on the particular browser you are using, as follows:

To display the JavaScript Console in Chrome:

Click the Chrome menu (in upper right area of browser window) > More tools > Developer tools > Console. <<<-- Please use this when debugging your programs

For more information about Chrome Developer Tools:

  • Documentation: https://developer.chrome.com/devtools/index
  • CodeSchool has a free course on Chrome DevTools at https://www.codeschool.com/courses/discover-devtools

Firefox

To display the Javascript Console click Tools, Web Developer, Web Console.

Internet Explorer (aka IE)

To display the various JavaScript options click Tools | Internet Options | Advanced. Also, look for some kind of icon in the status bar at the bottom of your browser window. You may need to click or double-click this to bring up the error message window.

Safari for Mac

  1. Click on Safari in the Menu Bar and select Preferences.
  2. Go to the Advanced tab and check the checkbox 'Show Develop menu in menu bar'.
  3. Close out Preferences.
  4. Click on Develop in the Menu Bar and select Show Error Console.
