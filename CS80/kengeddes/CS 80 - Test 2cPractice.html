CS 80  Test 2cPractice

Instructions:  

  1. This test is open book, open notes, open computer, open Internet.
  2. You shall not solicit help from any person.
  3. Name your HTML file test2cPractice.html

Total points for test: 100.

Write the HTML, CSS, and JS that meets the following requirements:

  1. Set the text color to your favorite color.
  2. Set the background color to a contrasting color (such as the complementary color).
  3. Add the following elements to the page:
    1. level 1 header
    2. image
    3. table with two rows and two columns. The headings for the two rows shall be DOB and Country.
  4. Write a JS script that does the following:
    1. Ask the user for her name. Set the the level 1 header to that value.
    2. Ask the user for the URL of a graphics file. Set the source of the image element to that URL.
    3. Ask the user for her date of birth and country of origin. Set the second column of the table to these values.
